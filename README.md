# pyspark-tutorial
Python Spark入门教程


## 依赖
* [Jupyter Notebook](http://jupyter.org/)
* [Apache Spark](https://spark.apache.org/)

## 如何运行
```
$ cd pyspark-tutorial
$ PYSPARK_DRIVER_PYTHON="jupyter" PYSPARK_DRIVER_PYTHON_OPTS="notebook" pyspark
```
